**API Rate limiter**

This project is about an API Rate limiter using the token bucket algorithm, it helps to reduce the load on the server based on the number of clients communicating with the server. 

Technologies used: Java, Springboot

Testing done using: mockito, jmeter, postman

Version control: git, github
